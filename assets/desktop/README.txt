Beans desktop notes

- Beans was built for Linux Mint 22.3 Cinnamon on VirtualBox.
- Install VirtualBox Guest Additions before running the installer.
- Firefox is the default research browser.
- Brave and Chrome are installed but not modified.
- Tor Browser finishes setup on first launch.
- SearXNG is local-only and starts when you run `beans-searxng-start`.
- Whisper will use NVIDIA acceleration if `nvidia-smi` is present and validation succeeds.
- Reboot or log out before expecting `vboxsf` group membership to work.
- Use a master VM or snapshot as baseline and perform research in disposable clones.
